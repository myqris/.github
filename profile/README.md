## Tentang Kami

**MyQRIS** adalah platform rekening bersama (escrow) berbasis blockchain yang dirancang khusus untuk melindungi transaksi freelance di Indonesia. Kami memastikan setiap pembayaran aman, transparan, dan tercatat secara permanen di blockchain.

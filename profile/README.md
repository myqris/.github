## Tentang Kami

**MyQRIS** adalah platform rekening bersama (escrow) berbasis blockchain yang dirancang khusus untuk melindungi transaksi freelance di Indonesia. Kami memastikan setiap pembayaran aman, transparan, dan tercatat secara permanen di blockchain.

### Kenapa MyQRIS?

- **Escrow Aman** — Dana klien disimpan aman hingga pekerjaan selesai
- **Blockchain Record** — Semua transaksi tercatat di Base L2 blockchain
- **Dispute Resolution** — Sistem mediasi yang adil untuk perselisihan
- **QRIS Payment** — Pembayaran mudah melalui QRIS
